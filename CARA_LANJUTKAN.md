# Panduan: Cara Melanjutkan Pekerjaan di Komputer Lain

Ikuti langkah-langkah ini jika Anda ingin membuka, mengedit, dan mensinkronkan proyek ini dari perangkat baru.

## 1. Persiapan di Komputer Baru
1.  Pastikan Anda sudah menginstal **Microsoft Edge** atau **Google Chrome**.
2.  Buka **GitHub** dan masuk ke akun `erykindisland`.
3.  Buka repositori: `https://github.com/erykindisland/undangan-digital-premium`.
4.  Klik tombol hijau **Code** -> **Download ZIP**.
5.  Ekstrak file ZIP tersebut ke folder di komputer baru Anda.

## 2. Membuka di Antigravity (Komputer Baru)
1.  Buka aplikasi **Antigravity**.
2.  Pilih **Open Folder** dan arahkan ke folder proyek yang baru saja Anda ekstrak.
3.  Kirim pesan ke asisten AI: *"Saya ingin melanjutkan proyek undangan digital ini."* 

## 3. Alur Kerja Sinkronisasi (Agar Selalu Update)
Karena kita belum menggunakan sistem `git push` otomatis yang rumit, gunakan alur kerja sederhana ini:

### Saat Anda Selesai Mengedit di Komputer A:
- Buka link upload GitHub: [**Klik di Sini**](https://github.com/erykindisland/undangan-digital-premium/upload/main)
- Tarik file yang baru saja Anda ubah (misal `index_crystal.html`) ke sana.
- Klik **Commit changes**.

### Saat Anda Pindah ke Komputer B:
- Sebelum mulai bekerja, buka GitHub.
- Klik nama file yang ingin dikerjakan (misal `index_crystal.html`).
- Klik **Download** atau salin isinya ke folder lokal Anda di Komputer B.
- Sekarang Anda siap mengedit versi terbaru!

## 4. Troubleshooting
- **Website tidak terupdate?**: Pastikan Anda sudah klik "Commit changes" di GitHub. Cloudflare butuh waktu sekitar 10-20 detik untuk deploy ulang.
- **Data Google Sheets hilang?**: Tenang, data ada di cloud Google. Di komputer mana pun Anda membuka webnya, data akan tetap ditarik dari Google Sheets yang sama.
